About Me

My name is Daniel Button. I have been building websites ever since elementry school when I got my first computer, a Compaq Presario, and built a website that played a music video that I uploaded from an Ace of Base CD belonging to my sister. But that was just for fun. I didn't get serious about programming until the spring of 2018 when I took a full-stack web development bootcamp offered at George Washington University. I did the web development program instead of either the UI/UX program or data science program because I enjoy the combination of both the creative thinking and logical problem solving involved in building web applications. Since completing the program, I have used the skills I acquired to build my own web apps, some of which are shown here.
